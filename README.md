# Preámbulo

Este proyecto se engloba dentro del metaproyecto de la aedm (asociación de estudiantes de diseño de madrid) “Terceros Espacios”, dentro del grupo de trabajo “Muebles de uso Libre”.

Dentro de ese proyecto ha habido una excisión para presentar una solución desde el ámbito académico. Para saber más, dirígete a [https://aedm.org.es/3espacios](https://aedm.org.es/3espacios).

# Motivos

La ESDM (Escuela Superior de Diseño de Madrid) se enfrenta a un panorama complejo. Por un lado, tenemos un edificio con un tamaño nimio. Es ¼ de lo necesario para los alumnos que hay.

En este contexto, nos encontramos con clases impartidas en espacios comunes, ausencia de espacios de trabajo cubiertos, falta de una biblioteca digna (la actual tiene 2m²), falta de comedor, y ni mencionemos la cafetería, que no existe. Por último, en toda la escuela, se palpa una sensación de abandono total por parte del alumnado con la administración pública.

La escuela no da abasto. El equipo directivo hace malabares para que cada año, a pesar de las circunstancias, el centro salga adelante. Incluso en estas circunstancias, salen premios de diseño de esta escuela.

El actual equipo directivo está dispuesto a muchas cosas, de hecho, hace pocos años, iniciaron un proyecto llamado “Entre Árboles” para crear nuevas aulas en el jardín, que ha quedado paralizado y ha ocasionado problemas legales a la actual directora.

En la ESDM no tenemos apenas espacio interior, pero tenemos jardín. Esta iniciativa busca crear en el jardín elementos para estar. Busca generar terceros espacios donde no los hay, de manera rápida, flexible y en replicables en cualquier lugar del mundo. Este proyecto no es solo un proyecto, es un manifiesto de como el diseño colaborativo y abierto puede moldear el espacio que habitamos y hacer del lugar hostil un lugar acogedor.

Como limitaciones impuestas, hemos abrazado el suprarreciclaje, que busca a partir de deshechos, fabricar elementos de alta calidad.

En nuestra escuela, tenemos bastantes restos del proyecto de “Entre Árboles”, usaremos estos elementos para generar estructura.

Usaremos mayormente los deshechos que se además se puedan adquirir en cualquier otra parte para que no se quede en un proyecto de una vez, sino en una iniciativa atemporal que perdure.
